# 📊 Employee Attendance & Workforce Analytics Dashboard

An interactive HR Analytics dashboard built in **Microsoft Excel** to analyze employee attendance, workforce performance, overtime, and work patterns. This project demonstrates how Excel can be used to transform raw attendance data into meaningful business insights through KPIs, PivotTables, PivotCharts, and Slicers.

Project Objective

The objective of this project is to help HR managers and business leaders monitor workforce performance by:

- Tracking employee attendance
- Measuring absenteeism and leave patterns
- Monitoring overtime hours
- Evaluating employee performance
- Analyzing work mode distribution
- Supporting data-driven HR decision-making
